Picture: Kopano Tlape/GCIS</span></figcaption></figure><div class="articleBodyMore" id="article-more-body" data-reactid=".xi9oe7tx14.1.d.0.0.0.5.1"><!-- C-ON- TEXT_CONTENT_START --><p>Kigali - Forty-four African countries have signed an agreement establishing a free trade area seen as vital to the continent's economic development, the head of the African Union said Wednesday.</p>

<p>&quot;The agreement establishing the CFTA (African Continental Free Trade Area) was signed by 44 countries,&quot; said Moussa Faki Mahamat, chairperson of the AU commission.</p>

<p>The creation of a free trade area - billed as the world's largest - comes after two years of negotiations, and is one of the AU's flagship projects for greater African integration.</p>

<p>However, the agreement will still have to be ratified at a national level, and is only due to come into force in 180 days.</p>

<p>The full list of countries which did not sign the agreement is not yet available, however Nigeria is a notable absentee after President Muhammadu Buhari pulled out of this week's launch in Rwanda saying he needed more time for consultations at home.</p>

            this.css(opts.property, '');

            return this;

            return this;

        matchHeight._groups.push({
            elements: this,
            options: opts

        // match each element's height to the tallest element in the selection
        matchHeight._apply(this, opts);

        return this;

                    if (style) {

                    verticalPadding += _parse($that.css('padding-top')) + _parse($that.css('padding-bottom'));

                $that.css(opts.property, (targetHeight - verticalPadding) + 'px');
            });
        });

        $hiddenParents.each(function() {

        }
    };

            if (windowWidth === _previousResizeWidth) {
                return;
            }
            _previousResizeWidth = windowWidth;
        }

        if (!throttle) {
            _update(event);
        } else if (_updateTimeout === -1) {

                _update(event);
                _updateTimeout = -1;
            }, matchHeight._throttle);
        }
    };

    $(matchHeight._applyDataApi);
