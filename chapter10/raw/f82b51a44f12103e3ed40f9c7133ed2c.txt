<h2 style="text-align: center;"><span style="font-weight: 400;">Telegram Battles for User Privacy</span></h2>
<p><span style="font-weight: 400;">Supreme Court Judge Alla Nazarova today rejected an appeal filed by Telegram, objecting to Russia’s Federal Security Services (FSB) obtaining the encryption keys to its platform. As a result, Telegram will be legally obligated to grant security forces access to the private messages and other data of its users, unless it can convince an appellate court to reverse the decision. Shortly after the ruling was made public, Telegram CEO Pavel Durov had this to say:</span></p>
<p><img class="aligncenter size-large wp-image-131652" src="https://news.bitcoin.com/wp-content/uploads/2018/03/pavel-durov-1024x179.png" alt="Court Orders Telegram to Hand Russia Its Encryption Keys" width="696" height="122" srcset="https://news.bitcoin.com/wp-content/uploads/2018/03/pavel-durov-1024x179.png 1024w, https://news.bitcoin.com/wp-content/uploads/2018/03/pavel-durov-300x52.png 300w, https://news.bitcoin.com/wp-content/uploads/2018/03/pavel-durov-768x134.png 768w, https://news.bitcoin.com/wp-content/uploads/2018/03/pavel-durov-696x122.png 696w, https://news.bitcoin.com/wp-content/uploads/2018/03/pavel-durov-1068x187.png 1068w, https://news.bitcoin.com/wp-content/uploads/2018/03/pavel-durov.png 1166w" sizes="(max-width: 696px) 100vw, 696px" /></p>
<p><span style="font-weight: 400;">The company finds itself in a difficult position. With the pre-sale for its own cryptocurrency underway, Telegram is seen as a flagbearer for a movement that prides itself on respecting user privacy, deploying encryption, and thwarting attempts at government interference. But as a Russian enterprise, Telegram is bound by the draconian laws that give the country’s security forces extensive surveillance powers.</span></p>
<h2 style="text-align: center;"><span style="font-weight: 400;">Putin Wants a Backdoor Because Terrorists</span></h2>
<p><span style="font-weight: 400;">In 2016, President Putin decreed that messaging services should provide a means of decryption for the benefit of its security services. Monitoring terrorists was the excuse cited. Russia is known for its human rights abuses and covert surveillance of journalists and dissidents, however, and it is hard to imagine that access to Telegram’s vast trove of user data would be used responsibly to identify terrorists only.</span></p>
<p><span style="font-weight: 400;">Western governments, including the UK and U.S., have found themselves engaged in an encryption war with tech companies, who insist that it is impossible to implement backdoors without weakening protocols. Create an entry point for state security, they insist, and it will be exploited by hackers and state-sponsored attackers.</span></p>
<p><img class="aligncenter size-full wp-image-131651" src="https://news.bitcoin.com/wp-content/uploads/2018/03/telegram-court.jpg" alt="Court Orders Telegram to Hand Russia Its Encryption Keys" width="800" height="533" srcset="https://news.bitcoin.com/wp-content/uploads/2018/03/telegram-court.jpg 800w, https://news.bitcoin.com/wp-content/uploads/2018/03/telegram-court-300x200.jpg 300w, https://news.bitcoin.com/wp-content/uploads/2018/03/telegram-court-768x512.jpg 768w, https://news.bitcoin.com/wp-content/uploads/2018/03/telegram-court-696x464.jpg 696w, https://news.bitcoin.com/wp-content/uploads/2018/03/telegram-court-630x420.jpg 630w" sizes="(max-width: 800px) 100vw, 800px" /></p>
<h2 style="text-align: center;"><span style="font-weight: 400;">Security Forces Plead Their Case</span></h2>
<p><span style="font-weight: 400;">In court, the FSB argued that they would not be violating user privacy by obtaining encryption keys, because they would still require a court order to extract the data of specific users. Following the hearing, Telegram’s legal team dismissed this claim, </span><a href="https://www.bloomberg.com/news/articles/2018-03-20/telegram-loses-bid-to-stop-russia-from-getting-encryption-keys" target="_blank" rel="noopener"><span style="font-weight: 400;">stating</span></a><span style="font-weight: 400;">: “The FSB’s argument that encryption keys can’t be considered private information defended by the Constitution is cunning. It’s like saying, ‘I’ve got a password from your email, but I don’t control your email, I just have the possibility to control.’”</span></p>
<p><span style="font-weight: 400;">Telegram is a Russian success story, led by the crypto-friendly Pavel Durov. He founded Telegram after a successful spell as CEO of VK, which grew to become Russia’s largest social network. Durov is also a major bitcoin holder, and Telegram’s own ICO to launch an integrated cryptocurrency and blockchain is already halfway towards its $1.7 billion goal. If Telegram can’t convince the Supreme Court to overturn today’s ruling, its only option may be to up tools and exit Russia altogether. </span></p>

 function dateLive() {
    
    // Create a timestamp for 'now'
    var now = Math.round((new Date()).getTime() / 1000)
    
    // Create an array of timestamped elements in the Big Grid published within the hour
    var timeStamps = document.getElementsByClassName('tstamp')

    // Iterate over each item in the array and update the timestamp
    for(var i = 0; i < timeStamps.length; i++) {
        
        // Variable to store when this story was published
        var publishedTime = timeStamps[i].dataset.ts
        
        // Calculate how many minutes ago the story was published
        var publishedAgo = (now - publishedTime) / 60 // Minutes ago
        
        publishedAgo = publishedAgo.toFixed(0)
        //console.log(publishedAgo)
        
        // Replace text element with real-time calculated element 
        if (publishedAgo < 2) {
          timeStamps[i].parentElement.childNodes[0].nodeValue = "1 min ago"
        } else if (publishedAgo < 60) {
          timeStamps[i].parentElement.childNodes[0].nodeValue = publishedAgo + " mins ago"    
        } else {
          timeStamps[i].parentElement.childNodes[0].nodeValue = "1 hour ago"
        }
    }
}
