document.addEventListener("DOMContentLoaded", function() {

	if(window.location.pathname.split( '/' )[1] == "news"){

    var searchForm = document.forms.search_search;

    var dropdowns = [ document.getElementById("itv-glob-nav-dropdown") ];

    var hasSearch = !!searchForm;
    var hasPlaceholderSupport = "placeholder" in document.createElement("input");

    function addEvent(el, type, handler) {
        if (!el) return;

        if (el.addEventListener) {
            el.addEventListener(type, handler, false);
        } else if (el.attachEvent) {
            el.attachEvent("on" + type, handler);
        }
    }

    function eventHandler(el) {
        return function(evt) {
            toggleDropdown(evt, el.parentNode);
        };
    }

    function toggleDropdown(evt, parent) {
        var className = parent.className;
        var classNameMatcher = /\b ? is-open\b/;

        for (i = 0; i < dropdowns.length; i++) {
            if (parent !== dropdowns[i].parentNode) {
                dropdowns[i].parentNode.className =
                    dropdowns[i].parentNode.className.replace(classNameMatcher, "");
            }
        }

        if (className.match(classNameMatcher)) {
            parent.className = className.replace(classNameMatcher, "");
        } else {
            parent.className = className + " is-open";
        }

        evt.preventDefault ? evt.preventDefault() : evt.returnValue = false;
    }

    if (hasSearch) {
        if (!hasPlaceholderSupport) {
            var searchInput = searchForm.q;
            var placeholderText = searchInput.getAttribute("placeholder");

            searchInput.value = placeholderText;

            addEvent(searchInput, "focus", function() {
                if (searchInput.value == placeholderText) {
                    searchInput.value = "";
                }
            });

            addEvent(searchInput, "blur", function() {
                if (searchInput.value == "") {
                    searchInput.value = placeholderText;
                }
            });
        }

        dropdowns.push(document.getElementById("itv-glob-search-toggle"));
    }

    var events = ["click", "touchstart"];
    var handler, d, i, e, j;

    for (i = 0; i < dropdowns.length; i++) {
        d = dropdowns[i];
        handler = eventHandler(d);

        for (j = 0; j < events.length; j++) {
            e = events[j];
            addEvent(d, e, handler);
        }
    }

    var element = document.getElementById("itv-header-page-fragment");

    if (!element) return;

    if (!!document.createElementNS &&
        !!document.createElementNS("http://www.w3.org/2000/svg", "svg").createSVGRect) {
            element.className = element.className + " svg";
    }

});

<div class="item__content-block item__content-block--markdown">
  <p>The European Union&#39;s ambassador to Russia is being recalled to consult with Brussels over the Salisbury nerve agent attack.</p>

<p>Dutch Prime Minister Mark Rutte said the diplomat was being recalled as a &quot;measure&quot; and the move was not a &quot;sanction&quot; against Moscow.</p>

<p>Prime Minister Theresa May has said the European Council is &quot;standing together&quot; over the Salisbury nerve agent attack.</p>

<p>Earlier, Mrs May won the backing of EU leaders as they accepted the only &quot;plausible explanation&quot; was that Russia was responsible.</p>

<p>Speaking after her address to the other 27 EU leaders over dinner, she said: &quot;I welcome the fact that the EU Council has agreed with the United Kingdom Government&#39;s assessment that it is highly likely that Russia is responsible for the attempted murder that took place on the streets of Salisbury and that there is no plausible alternative explanation.</p>

<p>&quot;The threat that Russia poses respects no borders and it is a threat to our values and it is right that here in the EU Council we are standing together to uphold those values.&quot;</p>

</div>
<div class="item__content-block item__content-block--markdown">
  <p>In something of a victory for Mrs May, the European Council President Donald Tusk said leaders of the other 27 EU member states agree with the UK that it is &quot;highly likely&quot; Russia is responsible for the attack.</p>

<div class="item__content-block item__content-block--markdown">
  <p>The UK is at a diplomatic impasse with Russia, sparked by the nerve agent poisoning of former Russian spy Sergei Skripal and his daughter in Salisbury.</p>

<p>Arriving in Brussels, Prime Minister Theresa May told ITV News: “Russia staged a brazen and reckless attack against the United Kingdom when it attempted the murder of two people in the streets of Salisbury.</p>

<p>&quot;I will be raising this issue with my counterparts today because it is clear that the Russian threat doesn&#39;t respect borders and indeed the incident in Salisbury was part of a pattern of Russian aggression against Europe and its near neighbours from the western Balkans to the Middle East.&quot;</p>

<p>On the first evening of the EU summit, a Downing Street spokesperson said: &quot;The Prime Minister provided the President and Chancellor with a detailed update on the investigation into the reckless use of a military nerve agent, of a type produced by Russia, on the streets of Salisbury…</p>

<p>&quot;The UK, Germany and France reaffirmed that there is no plausible explanation other than that the Russian state was responsible.</p>

<p>&quot;The leaders agreed on the importance of sending a strong European message in response to Russia&#39;s actions and agreed to remain in close contact in coming days.&quot;</p>

<div class="item__content-block item__content-block--markdown">
  <p>Russia is just one of the major topics to be worked out at the EU summit, with the Brexit transition deal looming large.</p>

<p>Mrs May is expected to tell EU leaders it is their duty to show &quot;energy and ambition&quot; in negotiating a post-Brexit trade relationship which will be good for both Britain and Europe.</p>

<p>On Friday leaders of the other 27 EU member states are expected to approve a draft deal on Britain&#39;s transition to Brexit, opening the door for talks on trade.</p>

<p>European Council president Donald Tusk raised doubts earlier this week over whether a deal reached by Brexit Secretary David Davis and chief EU negotiator Michel Barnier would be acceptable to all 27 - with Spain reported to be holding out to improve its hand over Gibraltar.</p>

<p>But a senior UK official said on Thursday evening that &quot;all the indications are positive&quot; that the transition deal will get the thumbs-up.</p>
